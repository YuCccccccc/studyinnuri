
Thymeleaf
    HTML 엔티티
        웹브라우저는 < 을 태그의 시작으로 인식한다. 
        < 을 태크가 아닌 문자로 표현할 수 있는 방법이 HTML 엔티티다.
        &gt; 혹은 &lt; 등이다.

    이스케이프 vs 언이스케이프 (default 이스케이프)
        이스케이프 : HTML에서 사용하는 특수문자 -(변경)-> HTML 엔티티

    내추럴 템플릿
        타임리프는 HTML 태그에 th:* 속성을 지정하는 방식으로 동작한다.
        템플릿 엔진이 동작하지 않고 그냥 파일로 열면 th:name 속성은 없는 속성으로 무시된다.
        서버에서 타임리프 템플릿이 동작되면 타임리프 속성이 html 속성을 대체하고, 없는 속성이면 속성이 추가된다.
        이러한 특성때문에 그냥 파일로 열어도 HTML 구조가 유지된다.
        
        <input type="text" name="mock" th:name="userA" />
            => (템플릿 엔진X) <input type="text" name="mock" />
            => (템플릿 엔진O) <input type="text" name="userA"/>

    변수 표현식 : ${ SpringEL }
    지역 변수 : 선언한 태그 안에서만 사용할 수 있는 지역변수를 생성할 수 있다.
    편의객체 : request, response, session, servletContext 편의객체를 제공하지만, 스프링 3.0부터 미제공.
    편의객체 2 : locale, param
    스프링빈 접근 : 타임리프에서 스프링빈 접근이 가능하다.
    URL 처리 : URL 처리 문법을 통해 path variable + query param을 표현할 수 있다. 절대경로(/hello), 상대경로(hello) 표현 가능하다.
    
    리터럴
        문자 리터럴 : 원칙상 문자는 ' 작은따옴표로 감싸야 한다.
    Elvis 연산자
        ${data}?: '데이터가 없습니다.'
    No 연산자
        _ 타임리프가 실행되지 않는 것처럼 동작한다. HTML의 내용 그대로 사용.
    
    속성추가
        th:attrappend, th:attrprepend, th:classappend
    checked 속성
        HTML에서는 checked 속성만 있으면, 체크박스가 체크 된다. 한마디로 checked="false"도 체크된다. 
        타임리프의 th:checked="false" 속성은 checked 속성 자체를 제거한다.
    반복
        List, 배열, Iterable, Enumeration, Map(Map.Entry) 를 반복에 사용할 수 있다.
        th:each="user, userStat : ${users}"의 userStat을 통해 반복 상태를 확인할 수 있다.
    조건부 평가
        조건에 부합하면 태그를 렌더링하고, 조건에 틀리면 태그가 렌더링되지 않는다. 아예 표시가 안된다.
    주석
        타임리프 파서 주석
            타임리프의 진짜 주석. 렌더링에서 주석부분을 제거한다.
        타임리프 프로토타입 주석
            파일로 열면 웹브라우저가 렌더링하지 않고, 타임리프 렌더링을 거치면 정상 렌더링이 된다. 
    블록
        타임리프 특성상 HTML 태그안에 속성으로 기능을 정의해서 사용하는데,
        태그 하나의 범위를 넘어서 태그들의 조합을 반복돌리거나 할 때 사용한다.
    
    자바스크립트 인라인
        타임리프에서 자바스크립트를 편리하게 사용할 수 있는 기능이다.
        자바스크립트 인라인을 사용하지 않으면 타임리프 렌더링 결과를 타입 상관없이 순수하게 해석해버린다.
        타입에 대한 마감처리와 기타 문법을 제공한다.

        자바스크립트 인라인 each
            자바스크립트 반복문을 돌릴 수 있는 문법이다.

    템플릿 조각
        HTML의 일부를 조각으로 분리해서, 원하는 곳에 삽입하는 기능이다.
        insert는 insert가 선언된 해당 태그 내에 삽입시키는 기능이고, 
        replace는 replace가 선언된 해당 태그 자체를 지우고 조각으로 대체한다.
    
    템플릿 레이아웃
        레이아웃 템플릿 기능으로, HTML 태그 자체를 파라미터로 전달할 수 있다.

    확장된 템플릿 레이아웃
        스프링 타일즈 같은 템플릿 레이아웃 기능이다.
